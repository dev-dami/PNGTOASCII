# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected Behavior

- Be kind and collaborative.
- Give actionable feedback.
- Assume good intent.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language
- Personal attacks
- Bad-faith disruption of project discussions

Maintainers may remove comments or contributions that violate this code.
